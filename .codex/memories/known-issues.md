# Known Issues

Use this file only for durable repo-level issues or failures that recur across sessions or branches.

## Promotion Rules

Keep an issue here only if at least one of these is true:

- it reproduces across multiple sessions
- it is not isolated to a single in-flight branch
- it materially affects normal development or debugging
- future sessions are likely to waste time rediscovering it

Do not promote branch-local work-in-progress failures here. Those belong in `active-branches/<branch>.md`.

## Entry Format

- Title
- Status
- Symptom
- Reproduction
- Likely surface area
- Last verified date
- Notes / linked artifacts
