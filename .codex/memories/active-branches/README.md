# Active Branch Handoffs

This directory stores volatile branch-scoped handoff files.

## Rules

- One file per active branch
- Filename should match the branch name: `<branch>.md`
- Keep each file compact and operational
- Update it at checkpoints and before ending a session
- Remove or archive it when the branch is merged and no longer active

## Required Companion

Use `_template.md` when creating a new branch handoff.
